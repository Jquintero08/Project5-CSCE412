My Portfolio

Reviewstar.tech

Reviewstar is a platform designed to help pinpoint problem areas in American Airline airports. To accomplish this website, we split up into two subteams: front end and back end. The back end team focused on the web-scrapping and processing that data via sentiment analyis to determine the most common complaints per airport. Front end team used HTML, CSS and Javascript to create the web app that displays our findings.

Building a Processor

During the time of creating this project, a partner and I were tasked with building a functioning processor through the use of Assembly and Verilog. We had to build different components of it such as the ALU, Control Unit, Sign Extender, DataMemory, and InstructionMemory. Once this was completed, we built a test bench to test the functionality of our processor and verified it was correct through the user of ARMv8 code to see if the proper registers were being accessed.
